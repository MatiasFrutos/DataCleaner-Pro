# DataCleaner Pro - Project Overview

DataCleaner Pro is a local desktop application built with Python to analyze, clean and export messy Excel, CSV and TXT files.

## Purpose

The main goal is to help developers, analysts, administrative teams and businesses prepare data before importing it into databases, CRMs, ERPs or reporting systems.

## Problem solved

Many business files contain:

- empty rows
- empty columns
- duplicated records
- inconsistent column names
- text with unnecessary spaces
- invalid emails
- inconsistent dates
- CSV files with wrong separators
- old XLS files exported from legacy systems

DataCleaner Pro reduces manual work and improves data quality before operational usage.

## MVP features

- Load Excel, CSV and TXT files
- Preview loaded data
- Analyze data quality
- Detect empty values and duplicates
- Validate emails and dates
- Clean data automatically
- Export clean files to XLSX or CSV
- Generate process reports
- Save application logs

## Target users

- Developers
- Data analysts
- IT consultants
- Administrative teams
- CRM/ERP operators
- Small and medium businesses
